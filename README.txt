Twitter Trends Scraper
A web scraper to fetch Twitter trends using Selenium, Express.js, and MongoDB.

Dependencies:
selenium-webdriver
mongodb
express
body-parser
uuid
Setup:
Install dependencies:

npm install
Replace your credentials: In app.js, replace the following details:

Username: Line 31: Replace '' with your Twitter username.
Email/Phone: Line 48: Replace '' with your email or phone.
Password: Line 66: Replace '' with your Twitter password.
Run the application:
node index.js
Access the web page: Go to http://localhost:3000 and click on the button to fetch trends.
